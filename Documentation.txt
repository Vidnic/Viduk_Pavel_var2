Это программа для перевода цветов между разными моделями: RGB, CMYK и HLS. Сделана на Python с помощью библиотеки PySide6. Чтобы запустить программу, нужен соответсвенно Python и установленная библиотека PySide6.

В программе есть три панели слева для разных цветовых моделей. RGB работает с числами от 0 до 255, CMYK и HLS от 0 до 100, а тон в HLS от 0 до 360. Справа показывается текущий цвет и его значения во всех моделях.

Менять цвета можно тремя способами: двигать ползунки, вводить числа в поля или выбрать цвет из палитры кнопкой Pick Color. Когда меняются значения в одной модели, остальные автоматически пересчитываются.

Конвертация работает по стандартным формулам. Из RGB в CMYK сначала считается черный цвет, потом остальные относительно него. В HLS тон считается по положению в цветовом круге, насыщенность и яркость из значений RGB.

Если вводятся значения за пределами допустимых, программа сама обрезает их до нужного диапазона.
